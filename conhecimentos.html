Programação

Processing

O Processing definitivamente ajuda muito no processo de desenvolvimento de interações, levando em consideração suas múltiplas bibliotecas e sua comunidade extremamente amigável. É a linguagem de programação que eu mais utilizo para as atividades da bolsa, pois ela simplifica o uso da programação para finalidades gráficas, quando comparada a outras linguagens.

Front End Developer

Tenho muito interesse em seguir carreira como Desenvolver Front End para Web. Com isso em mente, o pacote básico de HTML, CSS e JavaScript se faz indispensável. No momento, estou estudando React, uma das principais bibliotecas de código aberto com foco no desenvolvimento de páginas Web interativas, com o intuito de me profissionalizar nesta área.

Design

Web Design

Buscando associar meu conhecimento de ambas as áreas, o estudo de Web Design se tornou fundamental na minha graduação. Ele me permite desenvolver sites melhores, que levam em consideração os diversos aspectos gráficos envolvidos neste processo. Este conhecimento também serve como uma importante ponte comunicativa, visto que em praticamente nenhum contexto o programador trabalha sozinho, e nesse sentido, meu background de Design enriquece e facilita muito meu trabalho enquanto programador.

Modelagem 3D

Durante os semestres, acabei sempre me responsabilizado durante os projetos pela parte da modelagem 3D. Nesse sentido, me tornei bastante familiar com o uso do Rhinoceros, especialmente via Grasshopper, para construção e desenvolvimento de formas tridimensionais. Vale citar também as experimentações com o Blender, software que dentre tantas possibilidades, também tem a modelagem como uma das suas principais capacidades.
