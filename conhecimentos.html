Processing

O Processing definitivamente ajuda muito no processo de desenvolvimento de interações, levando em consideração suas múltiplas bibliotecas e sua comunidade extremamente amigável. É a linguagem de programação que eu mais utilizo para as atividades da bolsa, pois ela simplifica o uso da programação para finalidades gráficas, quando comparada a outras linguagens.

Front End Developer

Tenho muito interesse em seguir carreira como Desenvolver Front End. Com isso em mente, o pacote básico de HTML, CSS e JavaScript se faz indispensável. No momento, estou estudando React, uma das principais bibliotecas de código aberto com foco no desenvolvimento de páginas Web interativas, com o intuito de me profissionalizar nesta área.

React

Em um primeiro momento, ao começar a estudar React, não estava tão claro pra mim o motivo de aprender a utilizar essa tecnologia, sendo que era possível realizar aquilo com JS e CSS puro. Mas a partir do momento que comecei a desenvolver projetos a utilizando, ficou claro pra mim o seu potencial para escalabilidade e otimização em sites, especialmente sites maiores em que a componentização se torna essencial.

Countdown Persona 5 Royal

Após estudar React por algum tempo, me surgiu o questionamento de qual seria meu primeiro projeto com ele. Depois de pensar e pensar, decidi fazer algo que realmente me interessava, com um design bem original: uma página Web de contagem regressiva para o Lançamento do jogo Persona 5 Royal para diversas plataformas! O processo foi extremamente didático e rico. Apesar de simples, a aprendizagem que tive em utilizar Hooks, controlar States e manipular imagens (muitas e muitas imagens) foi fundamental para me iniciar no mundo do React. Pensando em como enriquecer a experiência do site, incorporei, em parceria com meu amigo e também programador Levi Rios, uma API de notícias que se atualiza a cada 8 horas, com um SearchTerm fixo para sempre trazer notícias relacionadas a série do jogo. Caso queira acessar o site e ver os códigos ( agora que o jogo já foi lançado ele mostra quantos dias se passaram desde o lançamento!) aqui está o link: https://lnkd.in/dDHWcSrK

Web Design

Buscando associar meu conhecimento de ambas as áreas, o estudo de Web Design se tornou fundamental na minha graduação. Ele me permite desenvolver sites melhores, que levam em consideração os diversos aspectos gráficos envolvidos neste processo. Este conhecimento também serve como uma importante ponte comunicativa, visto que em praticamente nenhum contexto o programador trabalha sozinho, e nesse sentido, meu background de Design enriquece e facilita muito meu trabalho enquanto programador.
